# Desafío Técnico Data Scientist para Mercado Libre 

En este repositorio se encuentran los siguientes archivos:

1. El PDF y el CSV de la consiga, simplemente por si alguien no lo tiene a mano.
2. La resolución del desafío técnico está en el notebook llamado **Fraud_prevention**. Ahí está el desarrollo de los puntos clave del challenge: 
      * Hipótesis: Introducción y desarrollo
      * Análisis y transformaciones del dataset
      * Modelos utilizados
      * Evaluación
      * Conclusión
3. Además, se pidió documentar la utilización de herramientas de AI, esto se encuentra en el PDF llamado "**Utilización de herramientas de AI**".
4. Y finalmente, el output del notebook fueron los archivos .pkl (*columnas_modelo, threshold y modelo_xgboost_fraude*) los cuales son necesarios si se quiere probar el modelo en un notebook distinto al del desarrollo, algo así como una "simulación del modelo en producción", el cual se puede probar en el notebook "**funcion_predecir_fraude**".
